Usando um link externo e criando outra aba.

Se voce observou bem, o link nos leva a outra pagina, nos TIRANDO desta pagina. Para clicar no link e fazer o usuario abrir uma outra aba seria necessario utilizar:
Tag 'a' href+ website + target=_blank + rel (relativo a) = external
